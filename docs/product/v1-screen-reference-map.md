# V1 Screen Reference Map

Use this map before building or revising any V1 screen. The numbered image files in `assets/recipee screens/` are the visual references for layout, styling, button hierarchy, and expected functionality. Do not regenerate or reinterpret these designs when the matching reference exists.

## Platform Intent

Family AI Kitchen is mobile-first for iOS and Android. The web build is used as a fast preview and QA surface unless a later product decision makes web a first-class V1 target.
